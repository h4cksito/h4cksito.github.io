CTF - Mr. Robot

En esta máquina de nivel principiante/intermedio, el objetivo era encontrar tres claves ocultas. Para ello, realicé una enumeración exhaustiva, explotación de WordPress y escalada de privilegios para obtener acceso root.

1. Enumeración Inicial

Escaneo con nmap mostró puertos abiertos: 22 (SSH), 80 (HTTP), 443 (HTTPS).

Accedí a la web principal, encontré una consola con temática Mr. Robot.

Descubrí robots.txt con referencias a fsocity.dic y key-1-of-3.txt.

Primera clave obtenida:

073403c8a58a1f80d943455fb30724b9

2. Ataque a WordPress

WordPress versión vulnerable localizada. Se utilizó un ataque de fuerza bruta contra el login con el diccionario fsocity.dic.

Credenciales válidas encontradas: elliot / ER28-0652.

3. Escalada de privilegios

Accedí vía SSH como elliot. Usé técnicas de escalada básicas: inspección de permisos sudo, SUID, scripts mal configurados.

Obtuve la clave final de root:

04787ddef27c3dee1ee161b21670b4e4

Conclusión

Máquina entretenida con una curva de aprendizaje muy buena para quien empieza en el pentesting web y privilege escalation.
